### 代码

用于过程请求时候的错误修饰器。

    eno = {&#039;code&#039;:0, &#039;msg&#039;:&#039;&#039;}
    
    def enoChk(eno):
        def wrapper(fn):
            def _wrapper(*args, **kwargs):
                if eno[&#039;code&#039;] != 0:
                    return eno
                return fn(*args, **kwargs)
            return _wrapper
        return wrapper

这个修饰器，可以带上参数，所以需要两层的结构
